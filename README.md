### Good to know

Message limits: all characters for one message ( content + all embeds characters ) -> 6000 characters
In embeds limits: 
| Field       | Limit                  |
|-------------|------------------------|
| title       | 256                    |
| description | 4096                   |
| fields      | Up to 25 field objects |
| field.name  | 256                    |
| field.value | 1024                   |
| footer.text | 2048                   |
| author.name | 256                    |
